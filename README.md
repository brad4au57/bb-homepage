# Brad Bales — Personal Webpage

This repository contains the source for my simple personal webpage showcasing my work as a QA Automation Engineer and JavaScript developer.

## Overview

- Title: Brad Bales · QA Automation & JavaScript
- Purpose: A lightweight, responsive single-page site with contact details and links to my GitHub.

## Live Preview

Open `index.html` (or `home.html` for the header/logo variant) in your browser to view the site locally, or you can visit http://bradbales.com to view live.

## Project structure

- `index.html` — The primary single-page site (HTML, CSS, and a small amount of JavaScript).
- `home.html` — Alternate entry that redirects from the index.html.
- `styles/` — Custom CSS (styles/home.css contains the .brand-logo rule).
- `js/` — Small JavaScript that updates the footer year.
- `assets/` — Logo images and other static assets. Current files include:
  - `bb-logo-black.png`
  - `bb-logo-mixed-transparent.png`
  - `bb-logo-white.png`

## Features

- Dark theme with a custom color palette
- Responsive layout optimized for desktop and mobile
- Hero section describing role and focus
- Contact block with email and a link to GitHub
- Small JavaScript snippet that updates the footer year dynamically
